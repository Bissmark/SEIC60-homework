Welcome

Welcome to the world of web development! Building a website can be a daunting task, but with the help of this template you will be able to work confidently and safely in the realms of HTML, JavaScript and CSS. This template provides you with a safe playground to practise your coding skills, with all the necessary elements included for you to get started on building a website. A script tag has also been added so that you can instantly begin experimenting with code.
